# Cookinator

A sleek Hebrew recipe manager built with Angular 21. Browse your cookbook, find recipes by available ingredients, and follow step-by-step cooking mode — all in a clean glassmorphism UI with light & dark themes.

**[→ Live Demo](https://your-username.github.io/cookinator-web/)**

---

## Features

- **Recipe Catalog** — browse, search, and manage your recipes
- **Ingredient Finder** — enter what's in the fridge, get matching recipes
- **Cooking Mode** — step-by-step guide with progress tracking
- **Create & Edit** — add recipes with ingredients, steps, and photos
- **Pearl / Midnight themes** — light and dark mode with a green-tinted glass aesthetic
- **RTL** — fully right-to-left Hebrew interface

## Tech Stack

- **Angular 21** · standalone components · signals
- **Reactive Forms** — custom dynamic form engine
- **Angular Material** (CDK)
- CSS custom properties + glassmorphism design system
